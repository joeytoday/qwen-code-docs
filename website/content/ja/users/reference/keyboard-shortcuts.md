# Qwen Code キーボードショートカット

このドキュメントでは、Qwen Code で利用可能なキーボードショートカットを一覧表示しています。

## 一般

| ショートカット                   | 説明                                                                                                                  |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | ダイアログと提案を閉じます。                                                                                          |
| `Ctrl+C`                       | 実行中のリクエストをキャンセルし、入力をクリアします。2回押すとアプリケーションを終了します。                         |
| `Ctrl+D`                       | 入力が空の場合にアプリケーションを終了します。2回押して確認します。                                                   |
| `Ctrl+L`                       | 画面をクリアします。                                                                                                  |
| `Ctrl+O`                       | デバッグコンソールの表示を切り替えます。                                                                              |
| `Ctrl+S`                       | 長い応答を完全に出力し、切り捨てを無効にします。端末のスクロールバック機能を使用して出力全体を表示できます。         |
| `Ctrl+T`                       | ツール説明の表示を切り替えます。                                                                                      |
| `Shift+Tab` (Windowsでは`Tab`) | 承認モードを循環します (`plan` → `default` → `auto-edit` → `yolo`)                                                    |

## 入力プロンプト

| ショートカット                                        | 説明                                                                                                                              |
| ----------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                   | 入力が空のときにシェルモードを切り替えます。                                                                                        |
| `?`                                                   | 入力が空のときにキーボードショートカット表示を切り替えます。                                                                        |
| `\` (行末に) + `Enter`                                | 改行を挿入します。                                                                                                                |
| `Down Arrow`                                          | 入力履歴を下方向に移動します。                                                                                                    |
| `Enter`                                               | 現在のプロンプトを送信します。                                                                                                    |
| `Meta+Delete` / `Ctrl+Delete`                         | カーソルの右側の単語を削除します。                                                                                                |
| `Tab`                                                 | 既存の候補がある場合、現在のサジェストを自動補完します。                                                                            |
| `Up Arrow`                                            | 入力履歴を上方向に移動します。                                                                                                    |
| `Ctrl+A` / `Home`                                     | カーソルを行の先頭に移動します。                                                                                                  |
| `Ctrl+B` / `Left Arrow`                               | カーソルを左に1文字分移動します。                                                                                                 |
| `Ctrl+C`                                              | 入力プロンプトをクリアします。                                                                                                    |
| `Esc` (ダブルクリック)                                | 入力プロンプトをクリアします。                                                                                                    |
| `Ctrl+D` / `Delete`                                   | カーソルの右側の文字を削除します。                                                                                                |
| `Ctrl+E` / `End`                                      | カーソルを行の末尾に移動します。                                                                                                  |
| `Ctrl+F` / `Right Arrow`                              | カーソルを右に1文字分移動します。                                                                                                 |
| `Ctrl+H` / `Backspace`                                | カーソルの左側の文字を削除します。                                                                                                |
| `Ctrl+K`                                              | カーソルから行末までを削除します。                                                                                                |
| `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`      | カーソルを左に1単語分移動します。                                                                                                 |
| `Ctrl+N`                                              | 入力履歴を下方向に移動します。                                                                                                    |
| `Ctrl+P`                                              | 入力履歴を上方向に移動します。                                                                                                    |
| `Ctrl+R`                                              | 入力/シェル履歴を逆方向に検索します。                                                                                             |
| `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F`    | カーソルを右に1単語分移動します。                                                                                                 |
| `Ctrl+U`                                              | カーソルから行の先頭までを削除します。                                                                                            |
| `Ctrl+V`                                              | クリップボードの内容を貼り付けます。クリップボードに画像が含まれている場合は、保存され、プロンプト内に参照が挿入されます。         |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`        | カーソルの左側の単語を削除します。                                                                                                |
| `Ctrl+X` / `Meta+Enter`                               | 現在の入力を外部エディタで開きます。                                                                                              |

## サジェスト

| ショートカット  | 説明                                   |
| --------------- | -------------------------------------- |
| `Down Arrow`    | サジェストを下に移動します。           |
| `Tab` / `Enter` | 選択したサジェストを受け入れます。     |
| `Up Arrow`      | サジェストを上に移動します。           |

## ラジオボタン選択

| ショートカット     | 説明                                                                                                          |
| ------------------ | ------------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` | 選択を下に移動します。                                                                                        |
| `Enter`            | 選択を確定します。                                                                                            |
| `Up Arrow` / `k`   | 選択を上に移動します。                                                                                        |
| `1-9`              | 数字で項目を選択します。                                                                                      |
| （複数桁）         | 9より大きい数字の項目については、連続して数字キーを押して対応する項目を選択してください。                     |

## IDE 統合

| ショートカット | 説明                                 |
| -------------- | ------------------------------------ |
| `Ctrl+G`       | IDE から受信したコンテキスト CLI を表示 |
