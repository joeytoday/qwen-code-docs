# AIでぐちゃぐちゃのデスクトップを整理しました、Qwen Code Cowork使用シーン大全

> 無料オープンソースのAI作業アシスタントで、ファイル整理、一括リネーム、ストレージ清理、日常タスク自動化をサポート。

ファイルを探すたびにゴミ山から宝を探すような感じで、確かに保存したはずなのに見つからない。さらにイライラするのは、数ヶ月に一度システムから「**ディスク容量不足**」という通知が届くことです。

整理すべきは分かっていますが——

**Qwen Code Cowork** を見つけるまでは、本当に作業を手伝ってくれるAIアシスタントです。

## Qwen Code Coworkとは？

簡単に言えば、**無料オープンソース**のデスクトップAIアシスタントで、Qwen Code SDKをベースに構築されています。

ただし、これは単なるチャットボックスではなく、本当に**手を動かして作業を手伝ってくれる**AIです：

| 能力 | 説明 |
|------|------|
| 📝 コード作成・編集 | あらゆるプログラミング言語をサポート |
| 📁 ファイル管理 | 作成、移動、リネーム、整理 |
| ⚡ コマンド実行 | ビルド、テスト、デプロイ |
| 💬 質問回答 | コードベースやその他の質問に対応 |
| 🔧 タスク自動化 | 自然言語で説明できることなら何でも |

最も重要な点は：**完全無料、オープンソースで管理可能、データはローカル保存**です。

比較すると、Claude CodeはMaxサブスクリプション利用者のみ利用でき、月額100-200ドルかかりますが、Qwen Code Coworkはゼロコストで同様の能力を利用できます。

## 使用シーン大全

以下は、実際に使用して発見した非常に実用的なシーンで、それぞれ具体的な操作デモを付けています。

### シーン1：ぐちゃぐちゃのデスクトップ/ダウンロードフォルダを整理

デスクトップとダウンロードフォルダはまさにデジタルゴミ場で、スクリーンショット、ドキュメント、インストーラー、一時ファイルが混ざっています。ものを探すたびに半日かかり、見るだけで嫌になりますが、手動で整理するのは時間がかかりすぎます。

**Coworkの手助け方法**：

直接伝えます：「デスクトップを整理して、ファイルタイプ別に分類して、スクリーンショットはスクリーンショットフォルダに、ドキュメントはドキュメントフォルダに、インストーラーはインストーラーフォルダに入れて」

Coworkは：

1. まずデスクトップをスキャンし、すべてのファイルをリストアップ
2. 整理案のプレビューを提示
3. 確認後、自動的に分類と移動を実行

 
<Callout>
    **ヒント**：
    Planモードを使用してAIに先に計画させ、案を確認してから実行すると、より安全に管理できます。
</Callout>

### シーン2：ファイルの一括リネーム（写真を日付順にリネーム）

携帯電話からエクスポートした写真のファイル名はめちゃくちゃで、IMG_0001、IMG_0002……いつ撮影したか全く分かりません。「2024-01-15_海辺の夕日」のような形式にリネームしたいですが、数百枚の写真を手動で変更するのは永遠にかかります。

**Coworkの手助け方法**：

伝えます：「このフォルダ内の写真を撮影日でリネームして、形式はYYYY-MM-DD_連番にして」

Coworkは：

1. 各写真のEXIF情報（撮影日）を読み取る
2. 指定された形式で新しいファイル名を生成
3. すべてのリネーム操作をプレビュー
4. 確認後、一括実行

### シーン3：ローカルストレージ容量をチェックし、大きなファイルを見つける

またコンピュータがディスク容量不足を通知してきましたが、容量が何に使われているか分かりません。クリーンアップしたいですが、重要なファイルを誤って削除するのが怖く、各種クリーンアップソフトを開くとプライバシーの安全性が気になります。

**Coworkの手助け方法**：

伝えます：「コンピュータのストレージ容量使用状況をチェックして、容量を最も占有している10個のフォルダとファイルを見つけ、内容を簡単に説明し、削除可能かどうか、削除による影響も教えて」

Coworkは：

1. ディスク使用状況をスキャン
2. 容量を最も占有しているファイルとフォルダをリストアップ
3. 安全に削除できるものを分析（キャッシュ、一時ファイルなど）
4. クリーンアップ提案を提示

<Callout type="error">
    **注意**：
    ファイル削除前に必ず確認してください！最初はゴミ箱に移動し、問題がないことを確認してから完全に削除することをお勧めします。
</Callout>

### シーン4：メモの統合とフォーマット

メモは至る所に散らばっています：WeChatお気に入り、メモ帳、Notion、Obsidian……それらを一箇所に統合したいですが、形式が統一されておらず、手動でコピー＆ペーストして形式を調整するのは面倒です。

**Coworkの手助け方法**：

伝えます：「このフォルダ内のすべてのdocxファイルを一つのMarkdownファイルに統合して、作成時間順に並べ、形式を統一し、ファイル名は二級見出しとして表示して」

Coworkは：

1. すべてのファイル内容を読み取る
2. 形式を識別して統一
3. 指定されたルールで並べ替えて統合
4. 形式が整ったMarkdownファイルを生成

### シーン5：ドキュメントの一括処理（PDF内容の抽出）

PDF資料をたくさんダウンロードしましたが、中のテキストを抽出してメモを取りたいですが、一つずつ開いてコピーするのは遅すぎます。中にはスキャン版のPDFもあり、コピーすると文字化けします。

**Coworkの手助け方法**：

伝えます：「このフォルダ内のすべてのPDFファイルの内容を抽出して、対応するmarkdownファイルとして保存して」

Coworkは：

1. フォルダ内のすべてのPDFを巡回
2. テキスト内容を抽出
3. 各PDFに対応するtxtファイルを生成
4. 処理結果を報告

### ソースコードからインストール（上級者向け）

ソースコードからビルドしたい場合は、まずBunをインストールする必要があります：

```bash
# Bunをインストール
curl -fsSL https://bun.sh/install | bash
exec /bin/zsh

# リポジトリをクローン
git clone https://github.com/QwenLM/qwen-code-examples

# ディレクトリに入ってインストール
cd qwen-code-examples/apps/qwen-cowork
bun install

# 起動
bun run dev
```

## 使用ヒント

### 1. Planモードを活用する

ファイル操作のような取り消しできないタスクには、先にAIに計画案を立てさせることをお勧めします：

```
「先にこのフォルダの状況を分析して、整理案を提示して、直接実行しないで」
```

案に問題がないことを確認してから、実行させます。

### 2. 作業ディレクトリを指定

セッション作成時に作業ディレクトリを指定すると、AIはそのディレクトリ内でのみ操作するため、より安全です。

### 3. 機密操作は確認が必要

Coworkは削除や移動などの機密操作に対して、先に確認を求めますので、誤操作の心配はありません。

## その他の可能性

上記に挙げたシーン以外にも、Qwen Code Coworkは以下をサポートします：

- 🔄 日常タスクの自動化（定期バックアップ、ファイル同期）
- 💻 コードプロジェクト管理（プロジェクト初期化、依存関係インストール、テスト実行）
- 📊 データ処理（CSV変換、データクリーニング、形式変換）
- 📝 ドキュメント生成（コードからドキュメント生成、README生成）
- 🔍 コードレビュー（コード品質分析、潜在的な問題の発見）

自然言語で説明できることなら、実現できます。
