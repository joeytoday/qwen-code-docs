# 快速开始

> 👏 欢迎使用 Qwen Code！

本快速入门指南将帮助你在几分钟内开始使用 AI 驱动的编码辅助。到结束时，你将了解如何使用 Qwen Code 完成常见的开发任务。

## 前提条件

请确保你已具备：

- 打开一个**终端**或命令提示符
- 一个用于练习的代码项目
- 一个 [Qwen Code](https://chat.qwen.ai/auth?mode=register) 账户

## 步骤 1：安装 Qwen Code

要安装 Qwen Code，请使用以下方法之一：

### NPM（推荐）

需要 [Node.js 20+](https://nodejs.org/download)，你可以使用 `node -v` 检查版本。如果尚未安装，请先安装 Node.js。

如果你已经[安装了 Node.js 20+ 或更高版本](https://nodejs.org/en/download/)：

```sh
npm install -g @qwen-code/qwen-code@latest
```

### Homebrew（macOS、Linux）

```sh
brew install qwen-code
```

## 步骤 2：登录你的账户

Qwen Code 需要一个账户才能使用。当你使用 `qwen` 命令启动交互式会话时，你需要登录：

```bash

# 首次使用时系统会提示你登录
qwen
```

```bash

# 按照提示使用你的账户登录
/auth
```

选择 `Qwen OAuth`，登录你的账户并按照提示确认。登录后，你的凭证将被保存，你无需再次登录。

> [!note]
>
> 当你首次使用 Qwen 账户认证 Qwen Code 时，系统会自动为你创建一个名为 ".qwen" 的工作区。这个工作区为组织中所有 Qwen Code 的使用提供集中的成本跟踪和管理。

> [!tip]
>
> 如果你需要重新登录或切换账户，请在 Qwen Code 中使用 `/auth` 命令。

## 步骤 3：开始你的第一个会话

在任何项目目录中打开终端并启动 Qwen Code：

```bash

# 可选
cd /path/to/your/project

# 启动 qwen
qwen
```

你将看到 Qwen Code 的欢迎界面，其中包含你的会话信息、最近的对话和最新更新。输入 `/help` 查看可用命令。

## 与 Qwen Code 对话

### 提出你的第一个问题

Qwen Code 将分析你的文件并提供摘要。你也可以提出更具体的问题：

```
explain the folder structure
```

你也可以询问 Qwen Code 关于其自身功能的问题：

```
what can Qwen Code do?
```

> [!note]
>
> Qwen Code 会在需要时读取你的文件 - 你无需手动添加上下文。Qwen Code 还可以访问其自身的文档，能够回答关于其功能和能力的问题。

### 进行你的第一次代码修改

现在让我们让 Qwen Code 做一些实际的编码。尝试一个简单的任务：

```
在主文件中添加一个 hello world 函数
```

Qwen Code 将会：

1. 找到合适的文件
2. 向你展示建议的更改
3. 请求你的批准
4. 进行编辑

> [!note]
>
> Qwen Code 在修改文件之前总是会请求许可。你可以批准单个更改，或为一个会话启用“全部接受”模式。

### 结合使用 Git

Qwen Code 让 Git 操作变得像对话一样简单：

```
我更改了哪些文件？
```

```
用描述性消息提交我的更改
```

你也可以提示进行更复杂的 Git 操作：

```
创建一个名为 feature/quickstart 的新分支
```

```
显示最近的 5 次提交
```

```
帮我解决合并冲突
```

### 修复错误或添加功能

Qwen Code 在调试和功能实现方面非常熟练。

用自然语言描述你想要的内容：

```
为用户注册表单添加输入验证
```

或者修复现有问题：

```
存在一个用户可以提交空表单的错误 - 请修复它
```

Qwen Code 将会：

- 定位相关代码
- 理解上下文
- 实现解决方案
- 如果有测试可用则运行测试

### 试用其他常见工作流

有许多方式可以使用 Qwen Code：

**重构代码**

```
将认证模块重构为使用 async/await 而不是回调函数
```

**编写测试**

```
为计算器函数编写单元测试
```

**更新文档**

```
在 README 中添加安装说明
```

**代码审查**

```
审查我的更改并提出改进建议
```

> [!tip]
>
> **记住**: Qwen Code 是你的人工智能结对程序员。像与一位乐于助人的同事交谈一样与它交流——描述你想实现的目标，它会帮助你达成目标。

## 基本命令

以下是日常使用中最重要的命令：

| 命令                  | 功能说明                                         | 示例                          |
| --------------------- | ------------------------------------------------ | ----------------------------- |
| `qwen`                | 启动 Qwen Code                                   | `qwen`                        |
| `/auth`               | 更改认证方式                                     | `/auth`                       |
| `/help`               | 显示可用命令的帮助信息                           | `/help` 或 `/?`               |
| `/compress`           | 将聊天历史替换为摘要以节省 Token                 | `/compress`                   |
| `/clear`              | 清除终端屏幕内容                                 | `/clear` (快捷键: `Ctrl+L`)   |
| `/theme`              | 更改 Qwen Code 视觉主题                          | `/theme`                      |
| `/language`           | 查看或更改语言设置                               | `/language`                   |
| → `ui [language]`     | 设置界面语言                                     | `/language ui zh-CN`          |
| → `output [language]` | 设置 LLM 输出语言                                | `/language output Chinese`    |
| `/quit`               | 立即退出 Qwen Code                               | `/quit` 或 `/exit`            |

完整的命令列表请参见 [CLI 参考](./features/commands)。

## 初学者的实用技巧

**明确具体地请求**

- 不要说："修复这个 bug"
- 而是说："修复登录 bug，用户输入错误凭据后看到空白屏幕"

**使用分步指令**

- 将复杂任务分解为步骤：

```
1. 为用户资料创建一个新的数据库表
2. 创建一个 API 端点来获取和更新用户资料
3. 构建一个网页，允许用户查看和编辑他们的信息
```

**先让 Qwen Code 探索**

- 在进行更改之前，先让 Qwen Code 了解你的代码：

```
分析数据库架构
```

```
构建仪表板，显示英国客户最常退货的产品
```

**使用快捷方式节省时间**

- 按 `?` 查看所有可用的键盘快捷键
- 使用 Tab 进行命令补全
- 按 ↑ 查看命令历史
- 输入 `/` 查看所有斜杠命令
