# Клавиатурные сокращения Qwen Code

Этот документ содержит список доступных клавиатурных сокращений в Qwen Code.

## Общие

| Сочетание клавиш               | Описание                                                                                                              |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | Закрыть диалоги и предложения.                                                                                        |
| `Ctrl+C`                       | Отменить текущий запрос и очистить ввод. Нажмите дважды для выхода из приложения.                                     |
| `Ctrl+D`                       | Выйти из приложения, если поле ввода пустое. Нажмите дважды для подтверждения.                                        |
| `Ctrl+L`                       | Очистить экран.                                                                                                       |
| `Ctrl+O`                       | Переключить отображение консоли отладки.                                                                              |
| `Ctrl+S`                       | Позволяет выводить длинные ответы полностью, отключая их усечение. Используйте прокрутку терминала для просмотра всего вывода. |
| `Ctrl+T`                       | Переключить отображение описаний инструментов.                                                                        |
| `Shift+Tab` (`Tab` в Windows)  | Переключение режимов одобрения (`plan` → `default` → `auto-edit` → `yolo`)                                            |

## Входной запрос

| Комбинация клавиш                                 | Описание                                                                                                                            |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                | Переключение режима оболочки при пустом вводе.                                                                                      |
| `?`                                                | Переключение отображения сочетаний клавиш при пустом вводе.                                                                         |
| `\` (в конце строки) + `Enter`                     | Вставка новой строки.                                                                                                               |
| `Стрелка вниз`                                     | Навигация по истории ввода вниз.                                                                                                   |
| `Enter`                                            | Отправка текущего запроса.                                                                                                          |
| `Meta+Delete` / `Ctrl+Delete`                      | Удаление слова справа от курсора.                                                                                                   |
| `Tab`                                              | Автозаполнение текущего предложения, если оно существует.                                                                           |
| `Стрелка вверх`                                    | Навигация по истории ввода вверх.                                                                                                  |
| `Ctrl+A` / `Home`                                  | Перемещение курсора в начало строки.                                                                                                |
| `Ctrl+B` / `Стрелка влево`                         | Перемещение курсора на один символ влево.                                                                                           |
| `Ctrl+C`                                           | Очистка поля ввода                                                                                                                  |
| `Esc` (двойное нажатие)                            | Очистка поля ввода.                                                                                                                 |
| `Ctrl+D` / `Delete`                                | Удаление символа справа от курсора.                                                                                                 |
| `Ctrl+E` / `End`                                   | Перемещение курсора в конец строки.                                                                                                 |
| `Ctrl+F` / `Стрелка вправо`                        | Перемещение курсора на один символ вправо.                                                                                          |
| `Ctrl+H` / `Backspace`                             | Удаление символа слева от курсора.                                                                                                  |
| `Ctrl+K`                                           | Удаление от курсора до конца строки.                                                                                                |
| `Ctrl+Стрелка влево` / `Meta+Стрелка влево` / `Meta+B` | Перемещение курсора на одно слово влево.                                                                                           |
| `Ctrl+N`                                           | Навигация по истории ввода вниз.                                                                                                   |
| `Ctrl+P`                                           | Навигация по истории ввода вверх.                                                                                                  |
| `Ctrl+R`                                           | Обратный поиск по истории ввода/оболочки.                                                                                          |
| `Ctrl+Стрелка вправо` / `Meta+Стрелка вправо` / `Meta+F` | Перемещение курсора на одно слово вправо.                                                                                        |
| `Ctrl+U`                                           | Удаление от курсора до начала строки.                                                                                               |
| `Ctrl+V`                                           | Вставка содержимого буфера обмена. Если буфер содержит изображение, оно будет сохранено, и ссылка на него будет вставлена в запрос. |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`     | Удаление слова слева от курсора.                                                                                                    |
| `Ctrl+X` / `Meta+Enter`                            | Открытие текущего ввода во внешнем редакторе.                                                                                       |

## Предложения

| Сочетание клавиш | Описание                                        |
| ---------------- | ----------------------------------------------- |
| `Стрелка вниз`   | Перемещение вниз по списку предложений.         |
| `Tab` / `Enter`  | Принять выбранное предложение.                  |
| `Стрелка вверх`  | Перемещение вверх по списку предложений.        |

## Выбор переключателя

| Ярлык              | Описание                                                                                                      |
| ------------------ | ------------------------------------------------------------------------------------------------------------- |
| `Стрелка вниз` / `j` | Переместить выбор вниз.                                                                                       |
| `Ввод`             | Подтвердить выбор.                                                                                            |
| `Стрелка вверх` / `k` | Переместить выбор вверх.                                                                                      |
| `1-9`              | Выбрать элемент по его номеру.                                                                                |
| (многозначный)     | Для элементов с номерами больше 9, нажмите цифры подряд, чтобы выбрать соответствующий элемент.                |

## Интеграция с IDE

| Горячая клавиша | Описание                                 |
| --------------- | ---------------------------------------- |
| `Ctrl+G`        | Просмотреть контекст CLI, полученный от IDE |
